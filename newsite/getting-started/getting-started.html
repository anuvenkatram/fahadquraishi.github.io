3. Setting up your theme

HTML

I have provided an index file for each of the layouts that I've created. Simply select one of the following html files and rename it to index.html

  • index-freelancer.html
  • index-app.html
  • index-agency-highres.html
  • index-agency-flat.html

CSS

All the CSS that I've created can be found in custom.css.

LESS

If you are familiar with LESS, you can change the default stylesheet to link to css/less/custom.less and modify the LESS code either client-side or server-side. See the following URL for instructions and more info:

lesscss.org

Imagery

Every resource that I have used to create the images can be found in the Resources folder. Unfortunately I cannot provide support on updating the images on your website, but these files will serve as a good foundation if you know how to use Adobe Photoshop.

6. Animations

Special thanks to Dan Eden for creating these effects. View site

To use the CSS3 Animations, see the instructions below.

Add the following classes to an element

.animated .[animation-name]

Examples

HTML

<img class="animated flash">...</img>

Javascript

$('img').addClass('animated bounceIn');

Class Cheatsheet

Any of the following classes can be used.

Attention seekers:

flash bounce shake tada swing wobble wiggle pulse

Flippers (currently Webkit, Firefox, & IE10 only):

flip flipInX flipOutX flipInY flipOutY

Fading entrances:

fadeIn fadeInUp fadeInDown fadeInLeft fadeInRight fadeInUpBig fadeInDownBig fadeInLeftBig fadeInRightBig

Fading exits:

fadeOut fadeOutUp fadeOutDown fadeOutLeft fadeOutRight fadeOutUpBig fadeOutDownBig fadeOutLeftBig fadeOutRightBig

Bouncing entrances:

bounceIn bounceInDown bounceInUp bounceInLeft bounceInRight

Bouncing exits:

bounceOut bounceOutDown bounceOutUp bounceOutLeft bounceOutRight

Rotating entrances:

rotateIn rotateInDownLeft rotateInDownRight rotateInUpLeft rotateInUpRight

Rotating exits:

rotateOut rotateOutDownLeft rotateOutDownRight rotateOutUpLeft rotateOutUpRight

Lightspeed:

lightSpeedIn lightSpeedOut

Specials:

hinge rollIn rollOut



For further instructions, please visit this site
